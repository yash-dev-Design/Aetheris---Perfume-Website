# Perfume Paradise

A modern, responsive e-commerce website for luxury perfumes. Discover your signature scent from top brands with an intuitive shopping experience.

## Features

- **Product Catalog**: Browse a curated selection of 36 premium perfumes from brands like Chanel, Dior, Gucci, and more
- **Interactive UI**: Smooth animations, modal product views, and responsive design
- **Shopping Cart**: Add items to cart with quantity management and local storage persistence
- **Contact Form**: Integrated contact form with validation
- **Mobile-First**: Optimized for all devices with a clean, accessible interface

## Technologies Used

- **HTML5**: Semantic markup and accessibility features
- **CSS3**: Custom properties, flexbox/grid layouts, and modern styling
- **Vanilla JavaScript**: DOM manipulation, event handling, and state management
- **Font Awesome**: Icons for enhanced UI elements
- **Google Fonts**: Playfair Display and Montserrat typefaces

## Getting Started

### Prerequisites

- Node.js (v14 or higher)
- npm (comes with Node.js)

### Installation

1. Clone the repository:
   ```bash
   git clone <repository-url>
   cd perfume
   ```

2. Install dependencies:
   ```bash
   npm install
   ```

3. Start the development server:
   ```bash
   npm start
   ```

4. Open your browser and navigate to `http://localhost:3000`

## Usage

- Browse perfumes in the product grid
- Click "View Details" to see more information in a modal
- Add items to your cart using the "Add to Cart" button
- Manage cart items in the slide-out drawer
- Use the contact form to get in touch

## Customization

The site uses CSS custom properties (variables) for easy theming. Key colors and settings can be modified in the `:root` selector in `index.html`.
